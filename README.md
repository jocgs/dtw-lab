# README

## Descripción del Proyecto
Este proyecto se encarga de limpiar datos para un modelo de machine learning. Su objetivo es preparar los datos de entrada para que sean adecuados para el entrenamiento de modelos, asegurando que estén libres de errores y en un formato adecuado.

## Cómo Correr el Proyecto
1. Asegúrate de tener Python instalado en tu sistema.
2. Clona este repositorio en tu máquina local.
3. Navega al directorio del proyecto.
4. Ejecuta el siguiente comando para instalar las dependencias:
   ```bash
   pip install -r requirements.txt
   ```
5. Para correr el script principal, utiliza:
   ```bash
   python main.py
   ```

## Cambios Realizados
- Se añadieron funciones para calcular la media y la moda de los datos limpiados, mejorando así la preparación de los datos para el modelo de machine learning.
